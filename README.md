# WeatherFit

Aplikacja mobilna (Expo/React Native) do sprawdzania pogody i sugerowania ubioru.

## Funkcje
- aktualna pogoda dla miasta,
- prognoza na kolejne dni,
- rekomendacje stroju (Outfit),
- zapis ustawień użytkownika lokalnie.

## Stack
- Expo + React Native
- TypeScript
- Expo Router
- Axios
- AsyncStorage

## Uruchomienie
```bash
npm install
npx expo start
```

## Struktura
- `app/` – ekrany
- `services/weatherApi.ts` – API pogody
- `services/settingsStorage.ts` – ustawienia
